# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the Changesets repository](https://github.com/changesets/changesets)

## How to add a changeset

1. Run `pnpm changeset` in the repo root
2. Select the packages you want to include (e.g. `@igrp/igrp-framework-react-design-system`)
3. Choose the version bump type (major, minor, patch)
4. Write a summary of the changes
5. Commit the generated file in `.changeset/`

## How to release

1. Run `pnpm changeset version` to update versions and generate CHANGELOGs
2. Run `pnpm build` to build the packages
3. Run `pnpm publish -r` (or use the package-specific release script) to publish
