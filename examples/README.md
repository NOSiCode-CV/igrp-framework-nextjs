# IGRP Design System Examples

Reference implementations for AI agents and developers.

| File | Description |
|------|-------------|
| form-basic.tsx | Basic form with IGRPForm, IGRPFormField, IGRPInputText |
| form-validation.tsx | Form with zod validation (min, max, enum) |
| datatable-basic.tsx | IGRPDataTable with ColumnDef |
| crud-page.tsx | List + create modal (IGRPPageHeader, IGRPDataTable, IGRPModalDialog, IGRPForm) |

Copy these into your app (e.g. `templates/demo`) or use as reference when generating code.
