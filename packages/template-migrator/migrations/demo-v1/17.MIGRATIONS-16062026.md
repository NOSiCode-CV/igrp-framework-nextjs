# Migration Guide — Sidebar menu search config (`17-sidebar-menu-search-config`)

## What changed

### Sidebar menu search opt-in (`showMenuSearch`)

`IGRPSidebarDataArgs` gains a new optional `showMenuSearch?: boolean` field.
When `true`, an inline search input appears above the sidebar menus; when
`false` or omitted (the default), the sidebar renders exactly as before.

The template config enables it:

```ts
// src/igrp.template.config.ts
getSidebarData: async () => ({
  ...
  showMenuSearch: true,  // ← new
}),
```

| File | Change |
|---|---|
| `src/igrp.template.config.ts` | Added `showMenuSearch: true` in `getSidebarData` |
| `next.config.ts` | Added clarifying comment on `turbopackRoot` and a commented-out alternative value |
