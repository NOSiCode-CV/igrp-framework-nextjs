# Migration Guide — Template resync catch-up (`21-template-resync-catchup`)

## What changed

This migration re-captures the current content of nine `demo-v1` template files and
records the current `next`/`react`/`react-dom` dep pins that had drifted from the
shipped migrations. There is no behavioral change — this is a drift-parity snapshot
authored so `check:drift` returns to GREEN after the security-hardening and
middleware-refactor commits that landed on `fix/deep-review-remediation` without a
corresponding migration.

The captured files cover the auth middleware (open-redirect hardening, backslash
rejection, sanitized callbackUrl), the IGRP layout server action, the `(igrp)` route
group layout, the build-config assembler, and the core `lib/` helpers (`auth.ts`,
`utils.ts`, `report-error.ts`, `dal.ts`), plus `next.config.ts`.
