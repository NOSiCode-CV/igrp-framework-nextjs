# IGRP Sidebar

## Components

- `IGRPSidebarProvider` – Root
- `IGRPSidebar` – Sidebar panel
- `IGRPSidebarTrigger` – Toggle button
- `IGRPSidebarInset` – Main content area
- `IGRPSidebarHeader`, `IGRPSidebarFooter`, `IGRPSidebarContent`
- `IGRPSidebarMenu`, `IGRPSidebarMenuItem`, `IGRPSidebarMenuButton`
- `useIGRPSidebar` – Hook for open/close state
