# IGRPTextarea API Reference

## Import

```tsx
import { IGRPTextarea, type IGRPTextareaProps } from '@igrp/igrp-framework-react-design-system';
```

## IGRPTextareaProps

Extends `IGRPInputProps`. Key props: `name`, `label`, `helperText`, `required`, `error`, `rows`, `inputClassName`.

## Example

```tsx
<IGRPTextarea name="bio" label="Bio" rows={4} placeholder="Enter bio..." />
```
