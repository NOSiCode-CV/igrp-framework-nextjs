# IGRP Calendar API

## Variants

| Component | Purpose |
|-----------|---------|
| `IGRPCalendarSingle` | Single date selection |
| `IGRPCalendarSingleTime` | Single date + time |
| `IGRPCalendarRange` | Date range |
| `IGRPCalendarRangeTime` | Range + time |
| `IGRPCalendarMultiple` | Multiple dates |
| `IGRPCalendarMultipleTime` | Multiple + time |

## Use when

You need a standalone calendar (not inside an input). For form fields, prefer IGRPDatePicker*.
