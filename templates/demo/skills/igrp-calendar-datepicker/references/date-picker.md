# IGRP Date Picker API

## Variants

| Component | Purpose |
|-----------|---------|
| `IGRPDatePickerSingle` | Single date |
| `IGRPDatePickerRange` | Date range |
| `IGRPDatePickerMultiple` | Multiple dates |
| `IGRPDatePickerInputSingle` | Input with calendar popover |

## Common Props

`name`, `label`, `required`, `placeholder`, `dateFormat`, `disabledPicker`, `disableBefore`, `disableAfter`, `helperText`.
