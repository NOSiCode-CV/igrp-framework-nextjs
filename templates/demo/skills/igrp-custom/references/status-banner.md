# IGRPStatusBanner

## Props

| Prop | Type | Description |
|------|------|-------------|
| `text` | `string` | Main status text |
| `badgeText` | `string` | Badge text |
| `color` | `IGRPColorVariants` | success, primary, destructive, etc. |
| `variant` | `IGRPColorRole` | soft, solid, outline |
| `badgeVariant`, `badgeColor` | - | Badge styling |
