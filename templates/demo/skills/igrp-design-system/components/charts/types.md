```ts
Record<string, string | number>
```

```ts
{ dataKey: string; name?: string; color?: string; stackId?: string }
```

- `size`, `height`, `width`
- `legendPosition`, `showGrid`, `showTooltip`
- `valueFormatter`, `labelFormatter`
- `stacked`, `hideAxis`, `hideXAxis`, `hideYAxis`
