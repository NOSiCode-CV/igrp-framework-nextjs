# IGRPDropdownMenu

## Import

```tsx
import {
  IGRPDropdownMenu,
  IGRPDropdownMenuTrigger,
  IGRPDropdownMenuContent,
  IGRPDropdownMenuItem,
  IGRPDropdownMenuSeparator,
  IGRPDropdownMenuGroup,
  IGRPDropdownMenuLabel,
} from '@igrp/igrp-framework-react-design-system';
```

## Structure

- `IGRPDropdownMenu` – Root
- `IGRPDropdownMenuTrigger` – Button that opens menu (use `asChild` with IGRPButton)
- `IGRPDropdownMenuContent` – Menu panel
- `IGRPDropdownMenuItem` – Menu item
- `IGRPDropdownMenuSeparator` – Divider
- `IGRPDropdownMenuGroup` / `IGRPDropdownMenuLabel` – Grouping
