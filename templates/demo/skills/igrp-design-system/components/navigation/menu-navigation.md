# IGRPMenuNavigation

## Import

```tsx
import {
  IGRPMenuNavigationProvider,
  IGRPMenuNavigation,
  useIGRPMenuNavigation,
  type IGRPMenuNavigationItem,
} from '@igrp/igrp-framework-react-design-system';
```

## IGRPMenuNavigationItem

`{ id, label, href?, icon?, children? }` – Nested items supported.

## Usage

Wrap with `IGRPMenuNavigationProvider`, render `IGRPMenuNavigation` with `items` prop.
