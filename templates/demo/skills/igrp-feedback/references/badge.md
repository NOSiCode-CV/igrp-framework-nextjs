# IGRPBadge

## Import

```tsx
import { IGRPBadge, igrpBadgeVariants } from '@igrp/igrp-framework-react-design-system';
```

## Props

`variant` (default, secondary, success, destructive, warning, outline), `color`, `dot`, `children`.

## Example

```tsx
<IGRPBadge variant="success">Active</IGRPBadge>
<IGRPBadge variant="destructive">Inactive</IGRPBadge>
```
